# BotecoDoLuis

Sistema de gestão para o Boteco do Luis, focado em controle de comandas, estoque e finanças.

## 🚀 Tecnologias
- React + TypeScript
- Vite
- Tailwind CSS
- Firebase (Auth, Firestore, Analytics)
- shadcn/ui
- motion (framer-motion)
- Recharts (Relatórios)

## 🛠️ Funcionalidades
- Gestão de Comandas em tempo real
- Controle de Estoque com categorias
- Fluxo de Caixa e Finanças
- Cadastro de Clientes e Fornecedores
- Relatórios de Vendas e Despesas
