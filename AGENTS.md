# 🤖 Ecossistema de Agentes - Boteco do Luis

Este projeto é desenvolvido por uma equipe de agentes especializados, coordenados pelo `Orchestrator`.

## 📋 Quadro de Especialistas

| Agente | Responsabilidade |
|--------|------------------|
| **Orchestrator** | Coordenação multi-agente, síntese de resultados e interface principal com o usuário. |
| **Product Strategist** | Alinhamento de visão de negócio, priorização de features e ROI. |
| **Frontend Specialist** | UI/UX, Design System (Tailwind), Acessibilidade e Performance Client-side. |
| **Backend Specialist** | Lógica de servidor, Integrações Cloud e Segurança de API. |
| **Database Architect** | Modelagem Firestore, otimização de índices e integridade de dados. |
| **Quality Engineer** | Estabilidade, prevenção de bugs e validação de fluxos. |
| **Security Auditor** | Auditoria de Firebase Rules e proteção de PII (Dados Pessoais). |
| **Performance Optimizer** | Otimização de bundle, Core Web Vitals e velocidade de resposta. |

## 🛠️ Como utilizar
Sempre que uma tarefa for solicitada, o **Orchestrator** assume o comando, analisa a necessidade e "convoca" os especialistas pertinentes. 
Exemplo: *"Como Frontend Specialist, vou refatorar a UI... mas antes, o Database Architect precisa validar a nova estrutura."*
