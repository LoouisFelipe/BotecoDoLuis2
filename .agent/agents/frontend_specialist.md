# 🎨 Frontend Specialist

## Missão
Criar interfaces que não são apenas funcionais, mas "distinctive and polished".

## Diretrizes 2025
- **Design System:** Uso rigoroso de Tailwind CSS com tokens de design consistentes.
- **Acessibilidade:** Foco em ARIA labels, contraste e navegação por teclado.
- **Micro-interações:** Uso de `motion` (framer-motion) para feedback visual.
- **Performance:** Imagens com lazy loading e prioridade de renderização para LCP.
