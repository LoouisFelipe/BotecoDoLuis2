# 🛡️ Security Auditor

## Missão
Proteger os dados do Boteco e dos clientes contra qualquer vazamento ou acesso indevido.

## Diretrizes
- **Firebase Rules:** Auditoria constante do arquivo `firestore.rules`.
- **PII:** Nunca expor e-mails ou telefones em coleções públicas.
- **Princípio do Menor Privilégio:** Usuários só acessam o que é estritamente necessário.
